# VibeShift Remote Recovery- Nomadic Balance Landing Page

## Overview
VibeShift is a boutique wellness agency for digital nomads, providing remote nutrition counseling and mental health coaching. This landing page promotes the `Nomadic Balance` program designed to help remote workers manage burnout and unhealthy eating habits while traveling.

## Requirements

1. Mobile-First Design
    - Fully responsive layout.
    - Optimized for slow travel connections.

2. Interactive Burnout Assessment ("Quick Check" Widget)
    - 3-question JavaScript widget.
    - Shows custom recommendations based on answers:
        -> "Nutrition Deep-Dive"
        -> "Stress Management Plan"

    - Fully functional with no console errors.

3. Hero Section
    - High-impact typography.
    - Clear CTA for a free 15-minute consultation.

4. Program Features
    - Grid or list layout highlighting three pillars:
        1. Cricadian Rhythm Alignment
        2. Cross-Border Nutrition
        3. Mental Resilience Coaching

5. Lead Capture Form
    - Styled contact form
    - Client-side validation:
        - Name: required
        - Email: required & proper format
        - Current Location: required
    - Simulated Submission (no backend required)

6. Tech Stack
    - HTML5
    - CSS3 or Tailwind
    - Vanilla JavaScript
    - Local image assets

---

## Deliverables
1. Codebase: A ZIP file containing all source files (HTML, CSS, JS, and local image assets).
2. Repository: A link to a public GitHub repository containing the code and a detailed README.md.
3. Live Demo: A link to the deployed site (hosted on Vercel, Netlify, or GitHub Pages).
4. Visual Proof: A short screen recording (MP4 or GIF) demonstrating the "Quick Check" widget logic and the responsive mobile view.
